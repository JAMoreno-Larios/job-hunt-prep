#job-hunt-prep

## Introduction

This project ingests the user's resume, curriculum vitae, and other documents
and provides a job posting URL so that Ollama can generate answers to 
job posting questionnaires i. e. "Why are you interested in this position?",
"List three reasons for applying", and the sort.

## Stack

For this project, I intend to use:
- Python
- FastAPI
- Ollama
- LangChain
- LangGraph
- ChromaDB
- Streamlit
